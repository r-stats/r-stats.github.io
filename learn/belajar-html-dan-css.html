Saya bukan orang yang belatar belakang programmer dan tidak begitu mengerti membangun dan mengembangkan website. Website ini saya bangun dengan full html. <code>

<dl>, <dt> dan <dd>
Digunakan untuk membuat daftar, tapi tidak ada bullet-nya.
